\begin{DoxyVerb}    MQC_IntegerGTScalar is a function that returns TRUE if an intrinsic integer is greater 
    than a MQC_Scalar.

    is greater than the real part of the MQC_Scalar and FALSE if the intrinsic integer is

    real part of the MQC_Scalar, the function returns TRUE if the imaginary part of 
    MQC_Scalar is less than zero and FALSE otherwise. \end{DoxyVerb}
